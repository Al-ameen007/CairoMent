This dataset has been collected from all the resources :
Nafsany (slang)
Facebook pages (slang)
Twitter API (slang)
Dataworld (slang)
SemEval (slang)
Reddit (standard arabic)
Crowdsourcing (slang) 

The dataset contains:
3600 rows : depression 
3400 rows : normal 

The dataset dialects
5000 rows : slang 
2000 rows : standard arabic
